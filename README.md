# mybrain-unified-system

## Short Description

Unified AI Operating System for knowledge, media, production, data warehousing, agents, specialized systems, memory, interfaces, and decision-making.

## Official Root

```text
C:\mybrain-unified-system\unified_ai_system
```

## Core Architecture

The system is built around:

- General Mind
- System Heart
- Data Warehouse
- Memory: Bronze / Silver / Gold
- Agents and Personas
- Specialized Brains and Systems
- Interfaces
- Registry, Repository, Reports, and Monitoring

## Main Rule

No direct execution.  
No raw file enters the General Mind directly.  
No direct Gold.  
No Legacy system runs directly.  
Everything passes through Warehouse, Heart, Reports, and Decision Gate.

## Core Flow

```text
Input
→ Gateway
→ Heart
→ Router
→ Dispatcher
→ Registry / Name Graph
→ Systems / Brains / Agents
→ Reports
→ Decision Gate
→ Memory / Repository / Registry
→ Interface Output
```
